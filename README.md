# Test
команды для командной строки git bash

переход в нужную папку
cd <название папки>

создание репозитория
git clone https

запрос статуса гит
git status

настройка
git config - global user.name "<ваше имя>"
git config - global user.email "<адрес почты>"

если есть новый файл или изменения каких-либо файлов, его нужно добавить в коммит
git add .

добавить инфо о внесенных изменениями
git commit -m 'изменение'

обновить локальные изменения с веб версией
git push
