# Test
команды для командной строки git bash

переход в нужную папку
cd <название папки>

создание репозитория
git clone https

запрос статуса гит
git status

настройка
git config - global user.name "<ваше имя>"
git config - global user.email "<адрес почты>"

если есть новый файл или изменения каких-либо файлов, его нужно добавить в коммит
точка означает «добавить все изменённые и неотслеживаемые файлы в подготовленные».
git add .

добавить инфо о внесенных изменениями
git commit -m 'сообщение о внесенном изменении'

обновить локальные изменения с веб версией
git push

добавить конкретный файл под версионный контроль
git add <имя файла>

контрольная сумма SHA-1 - это хеш-сумма изменений по алгоритму криптокрафического шифрования

удалить файл 
git rm "имя файла"
git rm "dzen1.txt"
после этого git push

команда, показывающая историю изменений
git log
